# Coding Standards

## General
- C++98 compatibility required
- Clear and consistent naming
- Comprehensive comments
- Error handling for all operations

## Naming Conventions
- Classes: PascalCase
- Methods: camelCase
- Variables: camelCase
- Constants: UPPER_CASE
- Private members: _prefix

## File Organization
- One class per file
- Include guards
- Consistent indentation
- Logical grouping of methods

## Documentation
- Class purpose
- Method descriptions
- Parameters
- Return values
- Error conditions
